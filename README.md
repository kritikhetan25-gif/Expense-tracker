# Expense-tracker
The Expense Tracker is a simple Java-based console application that helps users manage their daily expenses. It allows users to add, view, and analyze expenses efficiently through a menu-driven interface.

This project was developed as part of my B.Tech IT (3rd Semester) learning to strengthen core Java and problem-solving skills.

🚀 Features
➕ Add new expenses (date, description, amount)
📋 View all recorded expenses in tabular format
📊 Get expense summary:
Total expenses
Average expense
Highest expense

🧭 User-friendly menu-driven interface
⚡ Lightweight and easy to run
🛠️ Technologies Used
Java
Core Java Concepts
Arrays
Loops
Conditional statements
User input using Scanner
